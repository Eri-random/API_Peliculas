# Challenge Backend - Java Spring Boot (API) :rocket:

Desarrollo de API para explorar el mundo cinematográfico

### Capacidades Aplicadas 
- Arquitectura Spring MVC
- Patrón REST
- DTO
- Manejo de Excepciones 
- API basada en Roles
- Autenticación de usuarios con JWT
- Envio de Correos

### Dependencias
- Spring Web
- Spring Data JPA
- Spring Validation
- Lombok
- Spring Security
- JJWT
- Sendgrid

### Conexión
- MySQL
